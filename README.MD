# Chill Studios

## What is this?

Just an engine delivering relaxing tunes with HTML5 Audio API. You can change tracks and volume.

## Add sounds

Just add .ogg files to /assets/sounds/

## Is it working on mobile devices?

At least on Android. Try it.
